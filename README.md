Message Shortcuts:
  ':Exit' allows client to leave chat
  
  ':(' == '[Feeling Sad]' <br/>
  ':)' == '[Feeling Happy]' <br/>
  ':mytime' displays current time of sender <br/>
  ':+1hr' displays time of sender in one hour <br/>
